## Performance Targets

- **Input Latency**: < 5ms
- **Render Time**: < 2ms (full screen), < 0.5ms (dirty regions)
- **Parser Throughput**: > 50 MB/s
- **FPS Capability**: 500+
- **Memory Usage**: < 100MB

## Development

```bash
# Install dependencies
npm install

# Run tests
npm test

# Watch mode
npm run test:watch

# Coverage
npm run test:coverage

# Benchmarks
npm run benchmark
```

## Testing Philosophy

Test-Driven Development (TDD):
1. Write tests first
2. Implement minimal code to pass
3. Refactor
4. Repeat
